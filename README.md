# 🏪 Sistema Web - Bodega

Este proyecto es un sistema web para la gestión de inventario en una bodega. Permite controlar productos, entradas y salidas, stock mínimo y máximo, y generar reportes tipo Kardex.

---

## 🚀 Tecnologías utilizadas

- PHP
- MySQL
- HTML/CSS/JavaScript
- Bootstrap
  - CSS
